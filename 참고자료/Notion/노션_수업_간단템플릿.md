## 🎯 사용 방법

### 노션에서 사용하기

1. **템플릿 복사**
   - 위의 템플릿 중 원하는 것 선택
   - 전체 내용 복사

2. **노션에 붙여넣기**
   - 노션 페이지 열기
   - `Ctrl + V` (Windows) 또는 `Cmd + V` (Mac)
   - 자동으로 형식 변환됨

3. **내용 작성**
   - 각 섹션에 내용 입력
   - 필요 없는 섹션은 삭제

### 데이터베이스 템플릿으로 사용하기

1. **데이터베이스 생성**
   - `/table` 입력
   - "표 보기" 선택

2. **템플릿 만들기**
   - "새로 만들기" ▼ 클릭
   - "템플릿 추가" 선택
   - 템플릿 내용 붙여넣기
   - "기본값으로 설정" 선택

3. **사용하기**
   - 새 행 추가 시 자동으로 템플릿 적용
   - 내용만 채우면 됨!
