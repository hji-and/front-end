## 1. 시작하기 - 이름 정하기

### 📄 페이지 이름 추천

#### 메인 페이지 이름
- **🏠 학습 대시보드** (가장 추천) - 여러 데이터베이스를 모아놓는 메인 페이지
- **📚 수업 정리** - 간단하고 명확함
- **📅 일일 수업** - 매일 수업을 강조할 때

#### 페이지 구조 예시
```
🏠 학습 대시보드 (메인 페이지)
├── 📚 수업 일지 (데이터베이스)
├── 📋 과제 관리 (데이터베이스)
├── 📆 시험 일정 (데이터베이스)
└── 📝 복습 노트 (데이터베이스)
```

### 📊 데이터베이스 이름 추천

- **📚 수업 일지** (가장 추천)
- **📖 학습 일지**
- **📝 수업 노트**

## 2. 데이터베이스 생성

### 방법 1: `/table` 명령어 사용 (가장 쉬움) ⭐

1. 노션에서 새 페이지 생성
2. `/table` 입력
3. "표 보기" 또는 "Table view" 선택
4. 데이터베이스가 자동으로 생성됨
5. 데이터베이스 이름을 "📚 수업 일지"로 변경

### 방법 2: `/inline` 명령어 사용

1. 노션에서 새 페이지 생성
2. `/inline` 입력
3. "데이터베이스 - 인라인" 또는 "Database - Inline" 선택

### 방법 3: 전체 페이지 데이터베이스 만들기

1. 새 페이지 생성
2. `/database` 입력
3. "데이터베이스 - 전체 페이지" 또는 "Database - Full page" 선택
4. 데이터베이스 이름을 "📚 수업 일지"로 변경

## 3. 속성 설정

### 필수 속성(Properties)

| 속성명 | 타입 | 설명 |
|--------|------|------|
| 날짜 | Date | 수업 날짜 |
| 과목명 | Select | 수학, 영어, 과학, 국어 등 (필요한 과목 추가) |
| 교사명 | Text | 선생님 이름 |
| 태그 | Multi-select | #중요, #시험범위, #과제, #복습필요, #이해안됨 |
| 이해도 | Select | ⭐⭐⭐⭐⭐ (1~5점) |
| 과제있음 | Checkbox | 체크박스 |
| 과제내용 | Text | 과제 상세 내용 |
| 시험일정 | Date | 관련 시험 날짜 |

### 속성 추가 방법

1. 데이터베이스에서 "+ 속성 추가" 버튼 클릭
2. 속성명 입력
3. 타입 선택
4. 저장

## 4. 템플릿 만들기

### 🎯 목표
캘린더나 테이블 뷰에서 날짜를 클릭하면 자동으로 수업 정리 템플릿이 적용된 페이지가 열리도록 설정하기

### 단계별 설정

#### 1단계: 템플릿 생성

**방법 A: "새로 만들기" 버튼에서 템플릿 생성 (가장 쉬운 방법) ⭐**

1. **수업 일지 데이터베이스**에서 오른쪽 상단의 **"새로 만들기"** 버튼 옆의 **▼ (화살표)** 클릭
2. 드롭다운 메뉴에서 **"템플릿 추가"** 또는 **"Add a template"** 선택
3. 템플릿 이름을 **"📚 수업 정리"**로 설정
4. 아래 템플릿 내용을 복사하여 붙여넣기
5. 저장 완료!

**방법 B: 기존 페이지에서 템플릿 만들기**

1. 데이터베이스에서 **기존 행(페이지) 하나 클릭**하여 열기
2. 페이지 오른쪽 상단의 **"⋯" (점 3개)** 클릭
3. **"템플릿으로 저장"** 또는 **"Save as template"** 선택
4. 템플릿 이름 입력 후 저장

## 5. 템플릿 만든 후 다음 단계

### 1단계: 템플릿 내용 작성

현재 열려있는 템플릿 편집 화면에서:

1. **속성(Properties) 설정** (왼쪽에 보이는 속성들)
   - 필요하면 속성 추가/수정
   - 예: "과목", "이해도", "유형" 등

2. **템플릿 본문 작성**
   - 위의 템플릿 코드 복사해서 붙여넣기

3. **저장**: 자동 저장되거나 상단의 저장 버튼 클릭

### 2단계: 템플릿 페이지 닫기

템플릿 편집이 끝나면:
- **← (뒤로가기)** 버튼 클릭하거나
- 상단의 데이터베이스 이름 클릭하여 데이터베이스로 돌아가기

### 3단계: 기본값으로 설정하기

데이터베이스 화면으로 돌아온 후:

1. 오른쪽 상단의 **"새로 만들기"** 버튼 옆의 **▼ (화살표)** 클릭
2. 방금 만든 템플릿이 목록에 보임
3. 템플릿 옆의 **"⋯" (점 3개)** 클릭
4. **"기본값으로 설정"** 또는 **"Set as default"** 선택

✅ 이제 새 행을 추가하거나 날짜를 클릭하면 자동으로 이 템플릿이 적용됩니다!

### 4단계: 테스트하기

#### 방법 1: 새 행 추가로 테스트
1. 데이터베이스에서 **"새로 만들기"** 클릭
2. 템플릿이 자동으로 적용되어 열리는지 확인
3. 날짜 속성에 날짜 선택
4. 내용 작성 시작!

#### 방법 2: 캘린더 뷰에서 테스트
1. 데이터베이스 상단에서 **"Add a view"** 클릭
2. **"Calendar"** 선택
3. 날짜 속성을 "날짜"로 설정
4. 캘린더에서 **날짜 클릭**
5. 템플릿이 자동으로 열리는지 확인!

## 6. 뷰(View) 설정

### 캘린더 뷰 추가

1. 데이터베이스 상단에서 "Add a view" 클릭
2. "Calendar" 선택
3. 날짜 속성을 "날짜"로 설정
4. 이름을 "📅 캘린더"로 변경

**사용 방법:**
- 캘린더 뷰에서 **원하는 날짜 클릭**
- 자동으로 템플릿이 적용된 새 페이지가 열림
- 날짜는 자동으로 입력되어 있음 ({{date}} 변수 사용 시)

### 보드 뷰 추가 (과목별)

1. "Add a view" 클릭
2. "Board" 선택
3. Group by를 "과목명"으로 설정
4. 이름을 "📚 과목별"로 변경

### 필터 뷰 추가 (이해도 낮은 수업)

1. "Add a view" 클릭
2. "Table" 선택
3. Filter 추가: "이해도" is "⭐⭐" 또는 "⭐"
4. 이름을 "⚠️ 복습 필요"로 변경

### 다른 뷰들

- **갤러리 뷰**: 시각적으로 수업 내용 확인
- **리스트 뷰**: 간단한 리스트 형식
- **타임라인 뷰**: 시험 일정과 과제 일정 관리

## 7. 추가 데이터베이스 만들기

### 과제 관리 데이터베이스

1. 새 페이지 생성
2. `/table` 입력 후 "표 보기" 선택
3. 이름을 "📋 과제 관리"로 변경

#### 속성 설정:
- **과제명** (Title)
- **과목** (Relation) → "수업 일지" 데이터베이스 연결
- **마감일** (Date)
- **상태** (Select) - 미시작/진행중/완료
- **우선순위** (Select) - 높음/보통/낮음
- **완료일** (Date)

### 시험 일정 데이터베이스

1. 새 페이지 생성
2. `/table` 입력 후 "표 보기" 선택
3. 이름을 "📆 시험 일정"으로 변경

#### 속성 설정:
- **시험명** (Title)
- **과목** (Select)
- **시험일** (Date)
- **범위** (Text)
- **준비상태** (Select) - 미시작/준비중/완료
- **관련 수업** (Relation) → "수업 일지" 데이터베이스 연결

### 복습 노트 데이터베이스

1. 새 페이지 생성
2. `/table` 입력 후 "표 보기" 선택
3. 이름을 "📝 복습 노트"로 변경

#### 속성 설정:
- **복습일** (Date)
- **관련 수업** (Relation) → "수업 일지" 데이터베이스 연결
- **복습 내용** (Text)
- **이해도 변화** (Select) - 개선됨/동일/부족

## 8. 대시보드 만들기

1. 새 페이지 생성
2. 이름을 "🏠 학습 대시보드"로 변경
3. 아래 구조로 구성:

```
# 🏠 학습 대시보드

## 📊 오늘의 수업
[수업 일지 데이터베이스 - 오늘 날짜 필터]

## 📋 진행 중인 과제
[과제 관리 데이터베이스 - 상태: 진행중 필터]

## 📆 다가오는 시험
[시험 일정 데이터베이스 - 시험일 정렬]

## ⚠️ 복습 필요
[수업 일지 데이터베이스 - 이해도 낮은 것 필터]
```

### 데이터베이스 연결하기

1. 대시보드 페이지에서 `/linked` 입력
2. 연결할 데이터베이스 선택
3. 필터 설정 (선택사항)
4. 완료!

## 9. 활용 팁

### 📱 모바일에서 빠르게 작성하기

#### 간단 버전 템플릿:
```

- 

⭐ 이해도: /5

❓ 질문:
- 

📋 과제:
- 

💭 한 줄 정리:
```

### 🔄 주기적 관리

#### 매일
- 수업 후 즉시 정리 (5-10분)
- 이해도 체크
- 과제 확인

#### 주간
- 주간 리뷰 페이지 작성
- 이해도 낮은 수업 복습 계획
- 다음 주 과제 확인

#### 월간
- 월간 학습 리포트 작성
- 시험 결과 분석
- 학습 방법 개선점 도출

### 💡 고급 활용 팁

#### 1. 필터 활용
- 특정 과목만 보기
- 이해도 낮은 수업만 필터링
- 과제 있는 수업만 보기
- 시험 범위 수업만 보기

#### 2. 관계형 데이터베이스 연결
- 수업 일지 ↔ 과제 관리 연결
- 수업 일지 ↔ 시험 일정 연결
- 수업 일지 ↔ 복습 노트 연결

#### 3. 공식(Formula) 활용
- 수업 일지 개수 자동 계산
- 이해도 평균 계산
- 과제 완료율 계산

#### 4. 롤업(Rollup) 활용
- 과목별 수업 횟수 집계
- 과목별 평균 이해도 계산

## ❓ 자주 묻는 질문

### Q1: "Templates" 옵션이 메뉴에 없어요
**A:** 노션 버전에 따라 위치가 다를 수 있습니다. **"새로 만들기" 버튼 옆의 ▼ 화살표**를 클릭하면 템플릿 메뉴가 나타납니다.

### Q2: 날짜를 클릭해도 템플릿이 안 열려요
**A:** 데이터베이스 설정에서 템플릿을 "기본값"으로 설정했는지 확인하세요. "새로 만들기" 버튼 옆의 ▼에서 템플릿을 기본값으로 설정할 수 있습니다.

### Q3: 여러 템플릿 중 선택하고 싶어요
**A:** "새로 만들기" 버튼 옆의 ▼ 화살표를 클릭하면 템플릿 목록이 나타납니다. 원하는 템플릿을 선택하면 됩니다.

### Q4: 모바일에서도 작동하나요?
**A:** 네! 모바일 앱에서도 동일하게 작동합니다. 날짜를 탭하면 템플릿이 적용된 페이지가 열립니다.

### Q5: 인라인 데이터베이스가 안 만들어져요
**A:** `/table` 명령어를 사용하세요. 가장 빠르고 간단한 방법입니다.
