## 🎨 노션 사용 팁

### 1. 템플릿 버튼 만들기
1. 노션 페이지에서 `/template` 입력
2. 위의 "수업 정리" 템플릿을 복사
3. 템플릿 버튼에 저장
4. 매일 클릭 한 번으로 새 페이지 생성

### 2. 데이터베이스 뷰 설정
- **캘린더 뷰**: 날짜별로 수업 일지 확인
- **타임라인 뷰**: 시험 일정과 과제 일정 관리
- **보드 뷰**: 과목별로 그룹화하여 확인
- **갤러리 뷰**: 시각적으로 수업 내용 확인

### 3. 필터 활용
- 특정 과목만 보기
- 이해도 낮은 수업만 필터링
- 과제 있는 수업만 보기
- 시험 범위 수업만 보기

### 4. 관계형 데이터베이스 연결
- 수업 일지 ↔ 과제 관리 연결
- 수업 일지 ↔ 시험 일정 연결
- 수업 일지 ↔ 복습 노트 연결

## 💡 활용 예시

1. **시험 전 복습**: 이해도가 낮은 수업만 필터링하여 집중 복습
2. **과제 관리**: 과제 관리 데이터베이스에서 마감일 순으로 정렬
3. **성적 분석**: 시험 결과와 수업 이해도를 연결하여 분석
4. **학습 패턴 파악**: 태그를 활용하여 어떤 유형의 수업에서 어려움을 느끼는지 파악
