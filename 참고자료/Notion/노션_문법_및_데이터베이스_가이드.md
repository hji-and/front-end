## 1. 노션 기본 문법

### 📝 텍스트 서식

#### 제목 (Headings)

```
# 제목 1 (가장 큼)
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
###### 제목 6 (가장 작음)
```

**또는 명령어 사용:**
- `/heading` 또는 `/h1`, `/h2`, `/h3`

#### 강조 (Bold, Italic)

```
**굵게** 또는 __굵게__
*기울임* 또는 _기울임_
***굵게 기울임*** 또는 ___굵게 기울임___
~~취소선~~
```

#### 하이라이트

```
==하이라이트==
```

#### 인라인 코드

```
`코드`
```

#### 링크

```
[링크 텍스트](URL)
또는
URL을 붙여넣으면 자동으로 링크 생성
```

---

### 📋 리스트

#### 글머리 기호 목록 (Bullet List)

```
- 항목 1
- 항목 2
  - 하위 항목 1
  - 하위 항목 2
- 항목 3
```

**명령어:** `/bullet`

#### 번호 목록 (Numbered List)

```
1. 항목 1
2. 항목 2
   1. 하위 항목 1
   2. 하위 항목 2
3. 항목 3
```

**명령어:** `/numbered`

#### 체크박스 (To-do List)

```
- [ ] 미완료 항목
- [x] 완료된 항목
```

**명령어:** `/todo`

---

### 📑 인용구

```
> 인용구 내용
>> 중첩 인용구
```

**명령어:** `/quote`

---

### 💬 콜아웃 (Callout)

```
> 💡 팁: 이것은 콜아웃입니다
> ⚠️ 경고: 주의하세요
> ✅ 성공: 완료되었습니다
```

**명령어:** `/callout`

**이모지 예시:**
- 💡 팁
- ⚠️ 경고
- ✅ 성공
- ❓ 질문
- 📝 노트
- 🔥 중요

---

### 📊 구분선

```
---
또는
***
```

**명령어:** `/divider`

---

### 💻 코드 블록

#### 인라인 코드
```
`코드`
```

#### 코드 블록
````
```
코드 블록

````

**명령어:** `/code`

````

```javascript

```
````

수식 블록

## 2. 노션 명령어 (Slash Commands)

### 기본 명령어

#### 텍스트 관련
- `/heading` 또는 `/h1`, `/h2`, `/h3` - 제목
- `/text` - 일반 텍스트
- `/bulleted-list` 또는 `/bullet` - 글머리 기호 목록
- `/numbered-list` 또는 `/numbered` - 번호 목록
- `/to-do` 또는 `/todo` - 체크박스
- `/toggle` - 접기/펼치기 목록
- `/quote` - 인용구
- `/callout` - 콜아웃
- `/divider` - 구분선

#### 미디어 관련
- `/image` - 이미지
- `/video` - 비디오
- `/audio` - 오디오
- `/file` - 파일
- `/embed` - 임베드 (YouTube, Twitter 등)

#### 데이터베이스 관련
- `/table` - 표 데이터베이스
- `/board` - 보드 데이터베이스
- `/gallery` - 갤러리 데이터베이스
- `/list` - 리스트 데이터베이스
- `/calendar` - 캘린더 데이터베이스
- `/timeline` - 타임라인 데이터베이스
- `/database` - 데이터베이스 옵션 전체
- `/inline` - 인라인 데이터베이스

#### 고급 기능
- `/code` - 코드 블록
- `/equation` - 수학 공식
- `/template` - 템플릿 버튼
- `/button` - 버튼
- `/link` - 링크
- `/page` - 새 페이지
- `/comment` - 댓글

## 3. 데이터베이스 기초

### 🎯 데이터베이스란?

노션의 데이터베이스는 정보를 체계적으로 저장하고 관리하는 도구입니다. 엑셀의 스프레드시트와 비슷하지만 더 강력한 기능을 제공합니다.

### 📊 데이터베이스 타입

#### 1. 표 (Table)
- 행과 열로 구성된 전통적인 스프레드시트 형식
- 가장 많이 사용되는 형식
- **명령어:** `/table`

#### 2. 보드 (Board)
- 칸반 보드 형식
- 상태별로 그룹화하여 관리
- 프로젝트 관리에 적합
- **명령어:** `/board`

#### 3. 갤러리 (Gallery)
- 카드 형식으로 시각적으로 표시
- 이미지나 커버 사진이 중요할 때
- 포트폴리오, 제품 카탈로그에 적합
- **명령어:** `/gallery`

#### 4. 리스트 (List)
- 간단한 리스트 형식
- 제목과 몇 가지 속성만 표시
- 빠른 확인에 적합
- **명령어:** `/list`

#### 5. 캘린더 (Calendar)
- 날짜별로 표시
- 일정 관리에 적합
- 날짜 속성이 필수
- **명령어:** `/calendar`

#### 6. 타임라인 (Timeline)
- 시간 순서로 표시
- 프로젝트 일정 관리에 적합
- 시작일과 종료일 속성 필요
- **명령어:** `/timeline`

### 🔄 인라인 vs 전체 페이지

#### 인라인 데이터베이스
- 페이지 안에 삽입되는 데이터베이스
- 페이지 내용과 함께 사용
- **만들기:** `/table` 또는 `/inline`

#### 전체 페이지 데이터베이스
- 독립적인 페이지로 존재
- 전체 화면을 사용
- **만들기:** `/database` → "전체 페이지" 선택

## 4. 데이터베이스 속성(Properties)

### 📋 속성 타입

#### 1. 제목 (Title)
- 각 행의 고유한 이름
- 필수 속성 (자동 생성)
- 페이지로 열 수 있음

#### 2. 텍스트 (Text)
- 일반 텍스트 입력
- 짧은 메모, 설명 등

#### 3. 숫자 (Number)
- 숫자만 입력 가능
- 계산에 사용 가능

#### 4. 선택 (Select)
- 미리 정의된 옵션 중 선택
- 예: 상태 (진행중, 완료, 대기)
- 색상 태그 지정 가능

#### 5. 다중 선택 (Multi-select)
- 여러 옵션 선택 가능
- 예: 태그 (#중요, #시험범위)

#### 6. 날짜 (Date)
- 날짜 선택
- 시간 포함 가능
- 캘린더 뷰에서 사용

#### 7. 사람 (Person)
- 노션 워크스페이스의 사용자 선택
- 협업 시 담당자 지정

#### 8. 파일 및 미디어 (Files & Media)
- 파일, 이미지, 비디오 첨부
- 여러 파일 첨부 가능

#### 9. 체크박스 (Checkbox)
- 체크/언체크만 가능
- 완료 여부, 포함 여부 등

#### 10. URL
- 웹 링크 저장
- 자동으로 링크 생성

#### 11. 이메일
- 이메일 주소
- 클릭하면 이메일 앱 열림

#### 12. 전화번호
- 전화번호
- 클릭하면 전화 앱 열림

#### 13. 공식 (Formula)
- 다른 속성 값을 계산
- 수식 작성 가능
- 예: 합계, 평균, 조건문

#### 14. 관계 (Relation)
- 다른 데이터베이스와 연결
- 양방향 관계 설정 가능
- 예: 수업 일지 ↔ 과제 관리

#### 15. 롤업 (Rollup)
- 관계된 데이터베이스의 값 가져오기
- 집계 함수 사용 가능
- 예: 과제 완료율 계산

#### 16. 생성 시간 (Created time)
- 행이 생성된 시간
- 자동으로 기록

#### 17. 수정 시간 (Last edited time)
- 마지막 수정 시간
- 자동으로 기록

#### 18. 생성자 (Created by)
- 행을 만든 사람
- 자동으로 기록

#### 19. 수정자 (Last edited by)
- 마지막으로 수정한 사람
- 자동으로 기록

## 5. 데이터베이스 뷰(View)

### 🎨 뷰란?

같은 데이터베이스를 다른 방식으로 보는 것입니다. 여러 뷰를 만들어서 다양한 관점에서 데이터를 확인할 수 있습니다.

### 📊 뷰 타입

#### 1. 표 뷰 (Table View)
- 기본 스프레드시트 형식
- 모든 속성을 한눈에 확인
- 정렬, 필터링 쉬움

#### 2. 보드 뷰 (Board View)
- 칸반 보드 형식
- 그룹별로 카드 표시
- 드래그 앤 드롭으로 이동

#### 3. 갤러리 뷰 (Gallery View)
- 카드 형식으로 표시
- 이미지나 커버 사진 강조
- 시각적으로 보기 좋음

#### 4. 리스트 뷰 (List View)
- 간단한 리스트 형식
- 제목과 주요 속성만 표시
- 빠른 스캔에 적합

#### 5. 캘린더 뷰 (Calendar View)
- 날짜별로 표시
- 날짜 속성 필요
- 일정 관리에 최적

#### 6. 타임라인 뷰 (Timeline View)
- 시간 순서로 표시
- 시작일/종료일 속성 필요
- 프로젝트 일정 관리

### 🔍 필터 (Filter)

데이터베이스에서 특정 조건에 맞는 행만 보기

#### 필터 예시
- **과목이 "수학"인 것만 보기**
- **이해도가 "⭐⭐" 이하인 것만 보기**
- **과제가 있는 것만 보기**
- **오늘 날짜인 것만 보기**

### 🔄 정렬 (Sort)

데이터베이스 행을 특정 기준으로 정렬

#### 정렬 예시
- **날짜 순 (오름차순/내림차순)**
- **이해도 순**
- **과목명 순**
- **생성 시간 순**

### 📌 그룹화 (Group)

특정 속성 기준으로 그룹화

#### 그룹화 예시
- **과목별로 그룹화**
- **상태별로 그룹화**
- **날짜별로 그룹화**

## 6. 데이터베이스 고급 기능

### 🔗 관계 (Relation)

두 데이터베이스를 연결하는 기능

#### 예시: 수업 일지 ↔ 과제 관리
1. 수업 일지 데이터베이스에 "과제" 속성 추가
2. 타입: "관계" 선택
3. "과제 관리" 데이터베이스 연결
4. 양방향 관계 설정

**활용:**
- 수업 일지에서 관련 과제 바로 확인
- 과제에서 관련 수업 내용 확인

### 📊 롤업 (Rollup)

관계된 데이터베이스의 값을 가져와서 계산

#### 예시: 과제 완료율 계산
1. 수업 일지에 "과제 완료율" 속성 추가
2. 타입: "롤업" 선택
3. 관계된 "과제 관리" 데이터베이스 선택
4. 집계 함수: "완료된 과제 수 / 전체 과제 수"

### 🧮 공식 (Formula)

다른 속성 값을 계산하거나 변환

#### 공식 예시

**1. 날짜 차이 계산**
```
dateBetween(prop("시험일"), now(), "days")
```
- 오늘부터 시험일까지 남은 일수

**2. 조건문**
```
if(prop("이해도") == "⭐⭐⭐", "보통", "다시 공부")
```
- 이해도가 3점이면 "보통", 아니면 "다시 공부"

**3. 텍스트 결합**
```
prop("과목") + " - " + prop("단원")
```
- "수학 - 1단원" 형식으로 결합

**4. 숫자 계산**
```
prop("점수1") + prop("점수2") + prop("점수3")
```
- 세 점수의 합계

### 🎨 템플릿 (Template)

데이터베이스에 새 행을 추가할 때 자동으로 적용되는 형식

#### 템플릿 만들기
1. 데이터베이스에서 "새로 만들기" ▼ 클릭
2. "템플릿 추가" 선택
3. 템플릿 내용 작성
4. "기본값으로 설정" 선택

#### 템플릿 변수
- `{{date}}` - 날짜 속성 값
- `{{title}}` - 제목 속성 값
- `{{created}}` - 생성 시간

### 🔄 자동화 (Automation)

특정 조건에서 자동으로 작업 수행

#### 예시
- 매일 자동으로 새 수업 일지 생성
- 과제 마감일 3일 전 알림
- 이해도가 낮은 수업 자동 태그

## 7. 실전 활용 예시

### 📚 수업 일지 데이터베이스

#### 속성 설정
- **제목** (Title) - 수업 제목
- **날짜** (Date) - 수업 날짜
- **과목** (Select) - 수학, 영어, 과학 등
- **이해도** (Select) - ⭐⭐⭐⭐⭐
- **태그** (Multi-select) - #중요, #시험범위
- **과제** (Relation) - 과제 관리 데이터베이스 연결

#### 뷰 설정
- **캘린더 뷰**: 날짜별로 수업 확인
- **보드 뷰**: 과목별로 그룹화
- **필터 뷰**: 이해도 낮은 수업만 보기

### 📋 과제 관리 데이터베이스

#### 속성 설정
- **제목** (Title) - 과제명
- **과목** (Select) - 수학, 영어 등
- **마감일** (Date) - 과제 마감일
- **상태** (Select) - 미시작, 진행중, 완료
- **우선순위** (Select) - 높음, 보통, 낮음
- **관련 수업** (Relation) - 수업 일지 연결

#### 공식 예시
```
if(prop("마감일") < now(), "⚠️ 지연", 
   if(dateBetween(prop("마감일"), now(), "days") <= 3, "🔥 급함", "보통"))
```
- 마감일 지났으면 "⚠️ 지연"
- 3일 이내면 "🔥 급함"
- 그 외는 "보통"

### 📊 학습 통계 대시보드

#### 롤업 활용
- **과목별 수업 횟수**: 수업 일지에서 과목별 집계
- **평균 이해도**: 수업 일지에서 이해도 평균 계산
- **과제 완료율**: 과제 관리에서 완료율 계산

- **전체 페이지**: 독립적인 페이지, 전체 화면 사용

**A:**
- **관계**: 두 데이터베이스를 연결만 함
- **롤업**: 관계된 데이터베이스의 값을 가져와서 계산
